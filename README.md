# 🤰 Panduan Kehamilan Komprehensif

Website panduan kehamilan untuk calon orang tua (pemula total) — mobile-friendly, dipublikasikan via **GitHub Pages**.

🔗 **Live site:** `https://nicholaslie90.github.io/guide-kehamilan/` (setelah GitHub Pages aktif).

## Isi
- Perjalanan kehamilan per trimester
- Yang wajib dilakukan + jadwal kontrol (ANC) & USG
- Nutrisi: boleh & tidak boleh dikonsumsi
- Obat, skincare & aktivitas: aman vs hindari
- Tanda bahaya (kapan ke IGD)
- Persiapan finansial, administrasi & mental
- Persalinan normal vs caesar
- Hospital bag & perlengkapan bayi
- Rekomendasi merk produk + link Official Store
- Info RSIA Mahati & dr. Christian Sp.OG
